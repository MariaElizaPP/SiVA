Manual de uso

Como utilizar o sistema SiVA?

Para inserir um ponto de captação preencha os campos Nome da forma para preencher o nome conhecido daquela forma de captação, município, uf, o tipo de forma de abastecimento (ex.: Solução Alternativa Coletiva (SAC)) , além do tipo de captação (ex.: Subterranêa).

Para editar um ponto de captação selecione a linha a ser editada, preencha os campos "Nome da forma" para preencher o nome conhecido daquela forma de captação.

Para excluir um ponto de captação selecione a linha desejada e aperte o botão 'Excluir'.
